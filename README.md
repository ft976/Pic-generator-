# FLUX GEN - AI Image Generator

A beautifully crafted, modern, and highly responsive web application for generating high-quality images using FLUX.1 models (Schnell & Dev).

## ✨ Features

- **Conversational Interface:** Generate images seamlessly through a chat-like experience.
- **Multiple Models:** Switch between `FLUX.1 Schnell` (fast generation) and `FLUX.1 Dev` (higher quality).
- **Image-to-Image & Referencing:** Upload images to guide the generation process.
- **Local Persistence:** Your chat history and generations are safely stored in your browser's local storage so you don't lose any conversations.
- **Generation Controls:** Ability to cleanly stop generations midway with an interactive cancel button.
- **One-Click Downloads:** Download your generated artwork instantly.
- **Responsive Design:** A polished user interface that adapts flawlessly to desktop, tablet, and mobile devices, complete with a collapsible side menu.
- **Creator Profile:** Embedded developer details and useful quick-links inside the app profile section.

## 🛠️ Technology Stack

- **Frontend Framework:** React 18 with Vite
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **API Connectivity:** Integrated with backend generation services via Express/REST APIs.
- **Code Language:** TypeScript for robust and type-safe development.

## 🚀 Getting Started

1. Set up the necessary API keys in your environment (e.g., Gemini / Flux endpoints).
2. Install the application dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
